# NHDownloader (Beta)

Download your favorite NHentai doujinshis.

⚠️ **Warning: Early Beta Software**  
This extension is in active development. Things will break.

_Note: Please disable Internet Download Manager (IDM) extension when using NHDownloader extension to be able to create a custom file name for your doujinshis. Because if not, IDM extension will default it to "download.zip" every time._

## Features (Working)

- Download doujin as ZIP files with a nested folder with the same name as the ZIP file for easy extraction
- Get doujinshi title and ID as a default file name for download
- Image URL scrapping for current doujin page

## Known Issues

- See [CHANGELOG.md](CHANGELOG.md) for detailed bugs
- Error handling is minimal
- Download state is fragile

## Installation

1. Download latest release from GitHub
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the extension folder

## Reporting Issues

Please file bugs on GitHub issues page with:

- URL of the doujin page
- What you expected to happen
- What actually happened
- Screenshots if possible
